# NextJS Boilerplate

With :
- auth0
- chakra ui
- eslint / prettier / jest
- husky / lint-staged
- react-final-form / react-select
- react-query / axios

No getInitialProps, getServerSideProps.  

## Installation

```
yarn install
```

## Configuration

Create a `.env.local` file from `.env.sample` & set the constants.

## Run in dev

```
yarn dev
```
